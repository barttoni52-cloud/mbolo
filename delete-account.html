Supprimer mon compte

Gérez vos données et supprimez votre compte MBOLO

📱 Méthode 1 — Depuis l'application

La façon la plus rapide de supprimer vos données :

  1. Ouvrez MBOLO sur votre téléphone
  2. Allez dans l'onglet Profil
  3. Appuyez sur "Se déconnecter"
  4. Dans votre navigateur, allez dans Paramètres → Confidentialité → Effacer les données
  5. Sélectionnez "Données de site" pour le site MBOLO
  6. Confirmez la suppression

🗑️ Méthode 2 — Suppression manuelle des données

Puisque MBOLO stocke toutes vos données localement sur votre appareil, vous pouvez les supprimer directement :

  • Sur Android (Chrome) : Paramètres → Confidentialité et sécurité → Effacer les données de navigation → Données des sites
  • Sur iPhone (Safari) : Réglages → Safari → Effacer l'historique et les données

Cela supprime définitivement votre compte, vos avis et toutes vos données MBOLO.

⚠️ Attention : La suppression est définitive. Votre compte, vos établissements ajoutés et vos avis seront perdus et ne pourront pas être récupérés.

✉️ Demande de suppression par e-mail

Vous pouvez également nous envoyer une demande de suppression. Nous traiterons votre demande dans un délai de 7 jours.
